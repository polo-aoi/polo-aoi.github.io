# 我用 Claude 给自己写了个博客编辑器，顺便懂了啥叫 Vibe Coding

## 三四个月没更新主页，差点被现成工具“偷了家”

昨天心血来潮，想着个人主页已经三四个月没更新了，正好想把最近搞的几个软件更新上去。为了图省事，我下载了一个叫 Publii 的静态 CMS 工具。

听起来很完美对吧？但我按着教程配置完，点下那颗蓝色的“Sync”按钮后——**我的血压直接飙到了顶峰。**

短短几秒钟，我那纯手写、精心调校过排版的专属个人主页 `index.html`，被 Publii 霸道地全盘覆盖。看着满屏面目全非的老气模板，那种感觉，就像是你刚亲手给爱车做完精洗打好蜡，转头就被别人不由分说地泼了一桶劣质底漆。

我看着那一堆臃肿的配置项，彻底清醒了：我不需要邮件订阅。不需要 SEO 面板。更不需要一个自以为是的“黑盒”系统来褫夺我代码仓库的控制权。果断通过 GitHub 历史记录抢救回我原版的主页代码，删光垃圾文件，关掉那个蠢软件。

## 比上网吧还便宜的“极致正反馈”

其实回想一下，这半年我已经折腾了不少东西：
去年十二月，我完全用嘴指挥 Gemini，一点点搭出了现在这个个人网站；后来为了减轻自己摄影后期繁琐的排版工作，我又开发并持续更新了 AoiStitcher 拼图软件；最近迷上玩魔方，桥式进阶时搜到一个开源训练器觉得不好用，干脆自己动手改了一版。

一直以来，我都是个纯靠“用口指挥 AI”的野路子程序员。直到今天我才知道，原来这种脑子里有画面、嘴巴一说就能把东西捏出来的做法，叫 **Vibe Coding**。

这词真的太有意思了。为了努力跟上这个新时代，我现在每天固定花个三五块钱，跑 DeepSeek v4 模型的 token，研究 Skill、调教 Agent。我总是忍不住想：这几块钱的 token 花得简直太值了！比去网吧上网还便宜，但看着代码在眼前生成跑通，那种极速的正反馈带来的爽感，根本不是打几盘游戏能比的。

既然我已经习惯了让 AI 给我的想法打工，既然现成的博客工具都在教我做事，那我干嘛不自己造一个？打开 Claude，直接开干。

我跟它提了几个硬性需求：
- 本地跑一个编辑器，浏览器里写 Markdown，保留 GitHub README 编辑器那种纯粹的味道
- 侧边栏是文章列表，右边是编辑区，文章按分类组织，主页用手风琴折叠
- 图片拖进去自动压缩上传
- 写完一键推送到 GitHub Pages，换设备 clone 下来就能继续写
- **最重要的一点：绝对不许碰我原有的主页排版！**

没有 PRD，没有技术选型会。就是像平时那样聊着天，看着代码一行一行在我面前长出来。

## 这个专属编辑器的样子

### 沉浸式写文章
侧边栏点「写新文章」，右边出来编辑界面。标题、分类、标签、封面，都在最顺手的位置。正文写 Markdown，切到 Preview 实时看效果。图片拖进去自动压缩到 1600px、转 JPEG、85% 质量。自动保存到本地，1.5 秒防抖，写一半关浏览器也不怕。

### 极简分类管理
手风琴折叠的主页只有五个分类：开发日志、技术文章、摄影设计、生活随笔、水处理。没有任何多余的冗杂标签，编辑器里拖拽排序，分类改名，一切随心。

### 丝滑发布推送
写完点「发布并推送」——草稿变已发布，Python 脚本在后台无缝将内容转成 HTML 注入我的主页，自动 commit + push 到 GitHub。干净利落。

### 极致纯粹的技术栈
Flask + 原生 HTML/CSS/JS，零前端框架。Markdown 用 marked.js 渲染。数据全存文件，Git 仓库本身就是数据库。换台电脑 clone 下来 `python3 server.py` 直接开干。拒绝臃肿，只要极致的轻量与跟手感。

## 结语：跟紧科技的脚步

这不到半年的时间里，科技飞速进步带来的冲击感实在是太强烈了。

回想半年前用 Gemini 写网站的时候，我还觉得 AI 能写个网页已经很牛逼了。可这才五个月不到，AI 进化的速度简直离谱，现在 Claude 这种神级工具写出的代码质量、架构逻辑，比之前强了太多太多。

有幸能在这个时代亲身体验到这一切。从一开始看着屏幕上一堆完全听不懂的专业术语发懵，到现在跟着 AI 一步一步地去搞懂逻辑、去优化自己的工具，我真切地觉得，这种折腾对个人的成长有着莫大的好处。

每天花几块钱 token，张张嘴，AI 负责把砖砌好，你负责掌控全局。我根本不知道未来的科技还会进化成什么惊人的样子，但我真心希望自己能一直保持这份热情，紧紧跟上科技的脚步。
