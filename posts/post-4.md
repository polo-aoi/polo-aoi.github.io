# 网站架构升级日志 v2.0

<div class="meta">Published on Dec 12, 2025</div>

为了追求极致的性能与极简的维护体验，我对个人网站的底层架构进行了“降维打击”式的重构。

## 1. 架构重构：数据驱动 (Data-Driven)

放弃了传统的 HTML 硬编码模式，构建了基于 JSON 的数据层。现在，视图（View）与数据（Data）完全分离，新增文章只需配置数据，渲染引擎会自动生成精美的卡片。

## 2. 性能优化：懒加载 (Lazy Loading)

引入了 `loading="lazy"` 机制。图片资源按需加载，大幅降低了首屏网络请求，让移动端浏览体验如丝般顺滑。

## 3. 视觉系统：全局变量

使用 CSS 变量 (`:root`) 统一管理颜色。将“克莱因蓝”定义为全局参数 `--primary`。以后想换主题色，改一行代码，全站自动变色。

## 4. 逻辑增强：自动化

重写了 JavaScript 逻辑内核，实现了基于时间戳的**自动排序**与毫秒级的**全文检索**功能。让机器去处理逻辑，让人专注于创作。
